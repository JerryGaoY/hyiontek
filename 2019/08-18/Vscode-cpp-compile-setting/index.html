Windows下Visual Studio Code入门和配置C++编译环境

写在前面：

为什么选择 史上第一IDE Visual Studio Code（以下简称Vscode）?

  1. 支持多种语言的；
  2. 界面简洁，代码高亮清晰，自动补全功能完善；
  3. 插件商店提供多种插件，可满足多种需求；
  4. 用户多，支持完善……

1. 安装Vscode

不做赘述，传送门。

2. 设置简体中文

启动Vscode，点击右侧第五个图标打开扩展页(Extension)。

搜索”Chinese Simplified”，选择第一个扩展并安装。安装完，重启Vscode。完成设置中文。

3. 安装必要插件

同上，打开扩展页，搜索Code Runner,C/C++扩展并安装。

4. 安装编译包并配置环境

下载 MinGW（或 MinGW-w64 ）安装包。本文以 MinGW 为例。
下载链接：
MinGW
MinGW-w64

安装完成 MinGW 后，启动 MinGW Installation Manager ，如图勾选以下包。（勾选方法：单机复选框并点击 “Mark for installation” ）

然后，点击右侧 All Packages ，如图红线勾选必须项。其余可根据个人需要和下方注释勾选。

最后点击左上方 “Apply Changes” 安装。安装较慢，请耐心等待。（无法安装或过慢可以翻·墙）

安装完成后，到桌面，右击此电脑-属性-高级系统设置-环境变量-系统变量-Path-编辑-新建，然后输入刚才安装的 MinGW 的地址（一般在 C: 盘，如图：C:\MinGW\bin）

5. 最后配置

重启 Vscode，点击右侧“齿轮”-设置，在设置页中搜索 “run in terminal” ，找到 Code Runner: Run in terminal 项并勾选上复选框，即可完成配置。

6. 效果
